# Kick Space Technologies site

Next.js version of the Kick Space Technologies corporate site.

## Local development

```bash
npm install
npm run dev
```

Open http://127.0.0.1:3000.

## Production build

```bash
npm run build
```

This project uses `next build --webpack` so it can build reliably from the current Windows folder path.

## Vercel

Import this repository into Vercel as a Next.js project. The default build command is `npm run build`.
